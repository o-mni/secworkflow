Privacy & Data Notice

Please read before you begin.

Session Mode is the default. Data is held in memory only and cleared when you refresh or close this tab. Nothing is written to disk or transmitted anywhere — no analytics, no telemetry, no accounts.
Local Mode saves data in this browser. You can switch to Local Mode using the Session / Local toggle in the top bar. In Local Mode, data persists across browser restarts but is not encrypted and may be uploaded by browser sync (Chrome Sync, Firefox Sync, etc.).
Do not record passwords, private keys, tokens, or secrets. This tool is for workflow notes, checklist progress, and findings drafts — not a secrets vault.
To erase local data: use Data → Clear local data inside the app, or clear this site's data in your browser settings.
